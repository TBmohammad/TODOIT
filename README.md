# TODOIT

A modern and lightweight task management application built with Expo and React Native.

![Uploading photo_2026-07-03_22-55-16.jpg…]()

## Features

* Clean and intuitive interface
* Fast task creation and management
* Cross-platform support with Expo
* Open-source and easy to contribute

## Getting Started

```bash
npm install
npx expo start
```

## Download

Pre-built Android APK files are available in the **Releases** section of this repository. Download the latest release and install it directly on your Android device.

## Tech Stack

* React Native
* Expo
* JavaScript
